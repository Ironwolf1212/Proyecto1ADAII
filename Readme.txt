Para ejecutar el programa:
-Ejecutar cmd o powershell y dirigirse a la carpeta del proyecto
-Correr el archivo Interfaz.py con el comando: py interfaz.py

Para ejecutar una prueba:
-Presionar el botón de abrir archivo en la interfaz y seleccionar un archivo de entradas en el formato solicitado
-Confirmar la selección del archivo
-Elegir con qué algoritmo se desea resolver la entrada

Sobre los resultados:
-Después de seleccionar el algoritmo, la interfaz mostrará el resultado
-Además se guardará el resultado en un archivo resultado.txt con el formato de salida especificado
